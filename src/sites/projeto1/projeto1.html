HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo de páginas web. Ele introduz novos elementos e funcionalidades que facilitam o desenvolvimento de sites modernos, com foco em interatividade, multimídia e compatibilidade com dispositivos móveis.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é propriedade Display em CSS


A propriedade display em CSS controla como um elemento HTML é renderizado na página. Ela define como o elemento se comporta, seja como um bloco, inline, ou usando outros tipos de layout como flexbox ou grid. Em termos mais simples, ela determina se um elemento ocupa toda a largura disponível (bloco), se ele se comporta como parte de um texto (inline), ou se utiliza um sistema de layout mais avançado.

Display Características
Block: Em CSS, display: block faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e iniciando em uma nova linha. Elementos com display: block por padrão não permitem que outros elementos se alinhem ao seu lado, a menos que propriedades como float sejam usadas.
In-line: Um elemento com display: inline é exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo (o que é diferente de block, que vai ocupar todo o espaço disponível, mesmo que sobre espaço após o seu conteúdo).
In-line Block: Em CSS, display: inline-block combina características dos elementos inline e block. Ele permite que um elemento seja exibido na mesma linha que outros elementos (como inline), mas também aceita definir largura e altura (como block).
Flex: O display: flex no CSS define um elemento como um contêiner flexível, permitindo um layout mais flexível e responsivo para seus elementos filhos. É uma ferramenta poderosa para alinhar e distribuir espaço entre elementos de forma eficiente.
